## 1. LoRaWAN Gateway
- **Manufacturer:** Milesight
- **Product:** UG65 Indoor LoRaWAN Gateway
- **Official Link:** [Milesight UG65](https://www.milesight.com/iot/product/lorawan-gateway/ug65)

**Description:**  
A LoRaWAN gateway is used to connect many long-range, low-power LoRa devices to an IP network so that sensor data can be collected and forwarded to servers or cloud platforms. The Milesight UG65 is an 8-channel gateway based on the Semtech SX1302 LoRa chip and supports multiple backhaul options such as Ethernet, Wi-Fi, and cellular.

- **Product:** Anchor Beacon 2

## 7. Ultra-Wideband (UWB) Transceiver
- **Manufacturer:** Qorvo
- **Product:** DW3110
- **Official Link:** [Qorvo DW3110](https://www.qorvo.com/products/p/DW3110)

**Description:**  
A UWB transceiver is used for precise ranging, secure positioning, and short-range high-accuracy communication. The Qorvo DW3110 is based on IEEE 802.15.4z, supports data rates up to 6.8 Mbps, and can achieve positioning accuracy of less than 10 cm.
